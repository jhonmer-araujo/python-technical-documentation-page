Syntax

  • Identation

    Python uses indentation to indicate a block of code. Indentation refers to the spaces at the beginning of a code line. The indentation is very important.

    Example:

    if a > b: print("A is greater than B!") else: print("A is less than B!")

    Output while A is equal to 2 and B is equal to 1:

    A is greater than B!

    The following block will give an error, because skips the indentation.

    Example:

    if a > b: print("A is greater than B!") else: print("A is less than B!")

    Output error:

    IndentationError: expected an indented block

  • Multiple lines

    Statements in Python typically end with a new line. However, it allow the use of the line continuation character (\) to denote that the line should continue.

    Example:

    a = b + \ c + \ d + \ e

    Statements contained within the [], {}, or () brackets do not need to use the line continuation character.

    Example:

    number = ['one', 'two', 'three', 'four', 'five']

  • Quotations

    Python accepts single (') and double (") quotes to denote string literals. For using the string across multiple lines, Python accepts triple (''' or """) quotes.

    Example:

    name = 'James' fullname = "James Smith." address = """4455 Landing Lange, APT 4 Louisville, KY 40018-1234."""

  • Comments

    Comments start with a #, and Python will render the rest of the line as a comment.

    Example:

    #This is a comment. print("Comment example.") #End of comment

    Output:

    Comment example.

    Triple quotes can be used for multiline comment.

    Example:

    """ This is a comment. This is another comment. """

  • Multiple Statements in a same line

    The semicolon ( ; ) allows multiple statements on the single line given that neither statement starts a new code block.

    Example:

    name = 'James'; print("Hello, " + name)
Variables

These are some features of variables in Python:

  • A variable is created the moment of first assign a value to it. Python no needs command for declaring.
  • It Is possible assign a single value to several variables simultaneously.
  • Python has five standard data types; numbers, strings, lists, tuples, and dictionaries.
  • Variables are case sensitive.
  • Variables can be reassigned.
  • Variables can be combined with other variables.
  • String variables can be declared either by using single or double quotes
  • With type() Function, you can check the type of a variable.
  • It Is possible specify the data type of a variableh with casting; str(), int(), float()

Example:

a = 3 A = 4 b = "John" b = 'John' a = "Sally" # a is now of type string x = str(3) # x will be '3' y = int(3) # y will be 3 z = float(3) # z will be 3.0 print(type(a)) print(type(b))
